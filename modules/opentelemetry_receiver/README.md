# OpenTelemetry Receiver

Das folgende Modul stellt einen OpenTelemetry Receiver bereit.

## Argumente

Das Modul akzeptiert folgende Argumente:

| Name                     | default | Pflicht | Beschreibung                                                      |
| ------------------------ | :------ | ------- | ----------------------------------------------------------------- |
| otelcol_metrics_receiver | -       | Ja      | Endpunkt an den die extrahierten Metriken gesendet werden sollen. |
| otelcol_logs_receiver    | -       | Ja      | Endpunk an den die extrahierten Logs gesendet werden sollen.      |
| otelcol_traces_receiver  | master  | Nein    | Endpunkt an den die extrahierten Traces gesendet werden sollen.   |

## Exports

Zum aktuellen Zeitpunkt exportiert das Modul keine Werte.
